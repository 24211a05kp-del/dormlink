# General guidelines

* Only use absolute positioning when necessary. Opt for responsive and well structured layouts that use flexbox and grid by default
* Refactor code as you go to keep code clean
* Keep file sizes small and put helper functions and components in their own files.

--------------

# Design system guidelines

## Warm Theme (Current)
* Primary Color: `#5A3A1E` (Coffee/Deep Brown)
* Secondary Color: `#7A5C3A`
* Background: `#F5EFE6` (Beige/Warm White)
* Text Color: `#6B4F3A`
* Accent: `#EADFCC`
* Border Radius: `1rem` (2xl rounded) for cards and buttons.

## Typography
* Use a base font-size of 16px
* Use clean, modern sans-serif fonts (e.g., Inter, Geist)

## Components
### Button
* Primary Button: Filled `#5A3A1E`, white text.
* Secondary Button: Outlined `#5A3A1E`, text `#5A3A1E`.
* Hover effects: Subtle scale-up (105%) or background color shifts.

### Cards
* Use `bg-white` or subtle warm backgrounds.
* Add `shadow-md` or `shadow-lg` for depth.
* Padding should be consistent (e.g., `p-6` or `p-8`).
